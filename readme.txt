Fully featured access to content-distribution network BPI.


== Description ==

Wordpress BPI is the perfect solution for any wordpress website which needs to sync data with BPI-CDN.

* Syndication of content
* Push your own content to CDN BPI  (with or without images)
* Remove your own content from BPI
* Find and import content to your blog (with or without images)
* Field-mapping to improve use-ability
